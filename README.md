Graphical Template for NUHS
==========






A customizable Matplotlib-based visualization library styled with NUHS brand colors and fonts.



Features

currently, only various types of bar charts are added

- Bar chart
- Stacked bar chart
- Grouped bar chart
- Scatter plot
- Line plot
- Pie chart

[view demo](test_run.ipynb)
